## Getting Started

### Prerequisites

- Xcode 15 or newer (for iOS/macOS builds)  
- Android Studio Bumblebee or newer (for Android)  
- Node.js 18+ and npm/yarn (for Electron desktop)  

### Clone the repository

```bash
git clone https://github.com/DragoCam/WhyNote.git
cd WhyNote
```
